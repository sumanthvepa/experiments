Basic View Port Navigation
* Middle Mouse Button:   Rotate around  viewport orbit point
* Shift + Middle Mouse Button:  Pan the Viewport. This moves the viewport orbit point. Note that at least in blender 5.0, you have to unselect the selected cube once before paning. From then on selecting has no effect. 
* Scroll wheel: Zoom in/Zoom out towards/away from the view port orbit point

Adding Objects
Shift-A: Brings up the context menu to add an object Select the object that you want.

Note this brings up a context menu that allows you to modify the characteristics
of the object you just created. You can modify the parameters there. 

F9 - Opens the Last Operator Panel If for some reason you click away from the object bring up the last used context menu usijng F9. This only works if you have NOT
performed any other operation since the add. (This includes changing shading from
flat to smooth))
